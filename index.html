REFRESH

This is a simple browser-based application
that uses the Marvel Comics API in order
to search through the Marvel comic database.
It is designed by COMS W4170: User Interface
Design student Nishat Chowdhury.

1. Fill in the required input areas. Title can be any beginning keyword.

2. You will receive a list of a character image, comic book title, character name,
and character description. You may choose to order them as you wish.

3. Have FUN! If you have questions, click on the contact tab to communicate
about additional concerns.
