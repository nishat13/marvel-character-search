Marvel Query Search Page Using Marvel API.

    - There are input bars for title, start year, format, and order
        one desires the results to be set in.
    - User types in the input and hits submit and the application generates a list 
        of characters in each comic that matches user input as well as character
        thumbnail, name, and description. 
    - User is able to type in initial letters of a title, a start year, select the 
        issue format, and choose how the results are ordered by:  title, issueNumber, 
        or the default. Results display the name of the comic, name of the character,
        start year, and the thumbnail and description of the character.
